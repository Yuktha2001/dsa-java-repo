Arrays in Java – Complete Guide
1. What is an Array?
An array is a fixed-size, ordered collection of elements of the same data type, stored in contiguous memory locations. Instead of declaring separate variables for each value, you group them under one name and access each by its index.

2. Key Characteristics

Fixed size – once created, the size cannot change
Zero-indexed – first element is at index 0
Same type – all elements must be the same data type
Object in Java – arrays are objects and live on the heap

6. Accessing Elements

7. Array Length
javaint[] arr = {10, 20, 30, 40, 50};
System.out.println(arr.length);  // 5  (NOT a method, it's a property)

// Last element using length
System.out.println(arr[arr.length - 1]);  // 50

⚠️ Note: arr.length (no parentheses) — unlike String.length()

9. Common Array Operations
Find Sum & Average
javaint[] arr = {10, 20, 30, 40, 50};
int sum = 0;

for (int x : arr) sum += x;

double avg = (double) sum / arr.length;
System.out.println("Sum: " + sum);      // 150
System.out.println("Average: " + avg);  // 30.0
Find Max & Min
javaint[] arr = {3, 7, 1, 9, 4};
int max = arr[0], min = arr[0];

for (int x : arr) {
    if (x > max) max = x;
    if (x < min) min = x;
}
System.out.println("Max: " + max);  // 9
System.out.println("Min: " + min);  // 1
Reverse an Array
javaint[] arr = {1, 2, 3, 4, 5};
int left = 0, right = arr.length - 1;

while (left < right) {
    int temp = arr[left];
    arr[left] = arr[right];
    arr[right] = temp;
    left++;
    right--;
}
// arr is now {5, 4, 3, 2, 1}
Linear Search
javaint[] arr = {10, 25, 30, 45, 50};
int target = 30;
int index = -1;

for (int i = 0; i < arr.length; i++) {
    if (arr[i] == target) {
        index = i;
        break;
    }
}
System.out.println("Found at index: " + index);  // 2

10. Copying Arrays
javaint[] original = {1, 2, 3, 4, 5};

// Method 1: Manual copy
int[] copy1 = new int[original.length];
for (int i = 0; i < original.length; i++)
    copy1[i] = original[i];

// Method 2: System.arraycopy()
int[] copy2 = new int[5];
System.arraycopy(original, 0, copy2, 0, 5);
//              (src, srcPos, dest, destPos, length)

// Method 3: Arrays.copyOf()
import java.util.Arrays;
int[] copy3 = Arrays.copyOf(original, original.length);

// Method 4: Arrays.copyOfRange()
int[] copy4 = Arrays.copyOfRange(original, 1, 4); // {2, 3, 4}

⚠️ Shallow copy warning: int[] copy = original; does NOT copy — both variables point to the same array!

13. 2D Arrays (Matrix)
java// Declaration & Creation
int[][] matrix = new int[3][3];   // 3 rows, 3 columns

// Initialization
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Access element
System.out.println(matrix[1][2]);  // 6  (row 1, col 2)

// Traverse 2D array
for (int i = 0; i < matrix.length; i++) {          // rows
    for (int j = 0; j < matrix[i].length; j++) {   // columns
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}

14. Jagged Arrays (Irregular 2D Arrays)

jagged[1] = new int[4];   // row 1 has 4 elements
jagged[2] = new int[3];   // row 2 has 3 elements

};

15. Arrays as Method Parameters & Return Values
java// Passing array to a method
public static void printArray(int[] arr) {
    for (int x : arr) System.out.print(x + " ");
}

// Returning array from a method
public static int[] getArray() {
    return new int[]{1, 2, 3};
}

// Usage
int[] arr = {10, 20, 30};
printArray(arr);             // 10 20 30

int[] result = getArray();

Arrays are passed by reference — changes inside the method affect the original array.


16. ArrayIndexOutOfBoundsException
javaint[] arr = {1, 2, 3};

// ❌ This throws ArrayIndexOutOfBoundsException
System.out.println(arr[5]);  // index 5 doesn't exist!
System.out.println(arr[-1]); // negative index is invalid

// ✅ Always stay within bounds: 0 to arr.length - 1
```

## 17. Array vs ArrayList – Quick Comparison

| Feature | Array | ArrayList |
|---------|-------|-----------|
| Size | Fixed | Dynamic |
| Type | Primitive + Object | Object only |
| Performance | Faster | Slightly slower |
| Methods | Limited (`Arrays.*`) | Rich built-in methods |
| Syntax | `int[] arr` | `ArrayList<Integer>` |
```
